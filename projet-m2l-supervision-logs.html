Projet M2L - Supervision & Gestion des logs

BTS SIO SISR - IPSSI Marne-La-Vallée | Septembre 2025 - Juin 2026

Dans le cadre de mon BTS SIO option SISR à l'IPSSI, j'ai réalisé deux projets techniques pour la Maison des Ligues de Lorraine (M2L), une organisation disposant d'une infrastructure hétérogène composée de serveurs, machines virtuelles et équipements réseau.

Réalisation n°1
Mise en place d'une solution de supervision réseau Zabbix
Description

Déploiement d'une solution de supervision complète Zabbix sur l'infrastructure M2L afin de surveiller la disponibilité des équipements, mesurer les performances et générer des alertes en temps réel.

Tâches réalisées
  • Installation de Zabbix Server 7.x sur VM Debian 12 (avec MariaDB + Apache/PHP) hébergée sur Proxmox
  • Supervision de la VM OPNsense via SNMP (disponibilité, trafic réseau, interfaces VLAN)
  • Supervision du serveur Windows Server 2022 via agent Zabbix (CPU, RAM, disques)
  • Création de dashboards : Vue générale, Réseau, Serveurs
  • Configuration des alertes e-mail (seuils CPU, indisponibilité hôte)
  • Tests de validation : simulation de pannes et vérification des alertes
Technologies
Zabbix 7.x Proxmox Debian 12 MariaDB OPNsense Windows Server 2022 SNMP
Réalisation n°2
Mise en place d'une solution de gestion des logs Graylog
Description

Déploiement d'une solution centralisée de gestion des logs Graylog sur l'infrastructure M2L afin d'assurer le maintien en conditions opérationnelles et détecter les incidents de sécurité en temps réel.

Tâches réalisées
  • Installation de Graylog Server 5.x sur VM Debian 12 (avec MongoDB + OpenSearch) hébergée sur Proxmox
  • Création d'un input Syslog UDP 514 et configuration de la VM OPNsense pour envoyer ses logs vers Graylog
  • Création de dashboards : Vue générale, Sécurité (connexions bloquées, top IPs), VLANs
  • Configuration des alertes e-mail sur connexions bloquées
  • Tests de validation : génération de trafic bloqué et vérification des alertes
Technologies
Graylog 5.x Proxmox Debian 12 MongoDB OpenSearch OPNsense Syslog
